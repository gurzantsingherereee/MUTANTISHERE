## Fast replacement workflow

1. Open `assets/images/` in the repository.
2. Choose **Add file → Upload files**.
3. Upload the new PNG with the exact numbered filename.
4. Commit directly to `main`.
5. GitHub Pages will rebuild and publish the update automatically.
