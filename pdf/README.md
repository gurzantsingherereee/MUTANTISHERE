# WAVEX PDF files

Upload these three original PDF files into this folder using the exact filenames below:

- `WAVEX-5W-Packaging.pdf`
- `WAVEX-10W-Packaging.pdf`
- `WAVEX-User-Manual.pdf`

The live website checks these paths automatically. When a file is missing, its button shows **PDF not uploaded** instead of opening a broken 404 page. After the PDF is uploaded and GitHub Pages redeploys, the same button opens the document normally.
