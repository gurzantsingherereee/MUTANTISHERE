# WAVEX Display Font

Upload the licensed original display-font file into this folder using one of these exact names:

```text
WAVEX-Display.woff2   preferred
WAVEX-Display.woff
WAVEX-Display.ttf
```

The website already loads `WAVEX Display` through `@font-face` and applies it to hero headings, section headings, feature titles, packaging titles and the visual-language specimen.

The preferred production format is `WAVEX-Display.woff2` because it is smaller and loads faster.

Do not commit a font file unless you have permission to use and distribute it on the website.
