# Numbered website images

Upload your replacement images into this folder using the exact filenames:

```text
1.png
2.png
3.png
...
63.png
```

The live website automatically displays a temporary random image whenever a numbered PNG has not been uploaded yet.

When you add or replace a file, keep the same filename and commit the change. GitHub Pages will redeploy automatically.

See `/IMAGE-MAP.md` for the section, purpose and recommended dimensions of every number.
